# 运行程序

双击**start.vbs**即可运行程序（需要java运行环境）

----

# 表格的右键菜单

表格的右键菜单中是控制对应表格的操作

![表格右键菜单](http://fmn.rrimg.com/fmn063/20120808/0905/original_G7vx_43bd0000255f1260.jpg)

使用**添加\修改\删除**就可以方便的编辑任务了。

----

# 标签的右键菜单

标签的右键菜单中是控制整个程序的操作：

![标签的右键菜单](http://fmn.rrimg.com/fmn065/20120808/0905/original_Bqpn_376100001adf118c.jpg)

* **编辑**会弹出**终极编辑器**可以编辑任意年\月\周\日的任务
（主窗体中只能编辑今天\明天\本周\本月\本年的任务）
* **置顶**使得主窗体始终在其他窗体的上面
* **退出**会退出程序（点击关闭窗体按钮也可以）
* **最大\一般\最小**是本程序不同的显示模式

----

# 完成任务

今日表格（就是那个背景色与众不同的表格）有两列按钮：

* 左边的是**激活\中断**按钮
* 右边的是**完成**按钮

通过**激活\中断\完成**操作可记录一个任务的实际使用时间。
完成的任务可以被**隐藏已完成**隐藏，不能再进行任何操作。

----

# 开机启动

1. 创建start.vbs的快捷方式
1. 将start.vbs的快捷方式移动到**启动**文件夹中
	* Win7 在 开始 中
	* XP 在 开始->程序 中
